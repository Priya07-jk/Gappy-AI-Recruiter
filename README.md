#  AI-Powered Recruiter (Lemma Pod)

> Built for the **Ship to Get Hired - Gappy AI Hackathon**

An autonomous HR assistant powered by Lemma. This project aims to solve the massive bottleneck in modern hiring pipelines by instantly screening resumes, scoring candidates against job descriptions, and autonomously drafting personalized outreach emails.

##  The Problem
Recruiters spend over 70% of their time manually sifting through hundreds of resumes—many of which are unqualified or AI-generated spam. This manual screening delays time-to-hire and creates a poor candidate experience due to slow communication.

##  My Solution
I built a Lemma Pod that transforms hours of tedious manual screening into an instant, automated workflow:
1. **Ingest:** Drag and drop candidate resumes into the dashboard.
2. **Analyze:** The Lemma Agent parses the unstructured PDF data and scores each candidate against the specific job description.
3. **Structure:** Insights (skills, cultural fit, match score) are organized into a clean data table.
4. **Action:** The agent drafts highly personalized outreach emails and seamlessly connects to the recruiter's native email client to send them.

##  Built With
- **Lemma SDK:** The core cognitive engine. I utilized Lemma Pods, Agents, and Data Tables to handle the complex reasoning and data structuring.
- **React + Vite:** For the high-performance, glassmorphic frontend UI.
- **Custom CSS:** Tailored for rich aesthetics and micro-animations to create a premium user experience.

##  Running Locally

1. Clone the repository
2. Navigate to the app directory:
   ```bash
   cd app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
